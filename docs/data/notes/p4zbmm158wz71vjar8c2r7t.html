UI布局

页面布局

来一起看一下 Dendron 用户界面（工作台）长什么样子吧

🎗️ 小提示：如果你对 VS Code 还不太了解，接下来将介绍关于 VS Code 的基本知识。需要快速执行命令时（包括 Dendron 命令）需要用到命令面板。使用快捷键Ctrl+Shift+P/Cmd+Shift+P打开命令面板并输入命令，按回车键执行。如果你在接下来的指南中看到运行一个命令，请打开命令面板输入命令并执行。

🎗️ 小提示：VS Code 允许你重新设计这些窗口，您可以自由组合成最适应您的 Dendron UI 布局。详情查看VS Code documentation for more information on the VS Code UI

1. 编辑区域

在这里可以编辑您的笔记，在 Dendron 中笔记时Markdown文件(my.note.md)。

2. 预览界面

显示当前笔记的预览界面。你可以使用命令Dendron: Toggle Preview来打开和关闭预览界面。
在预览界面的最下面您可以看到[孩子]和[连接]两个部分，这部分内容在接下来的介绍。

🎗️ 小提示：预览界面是只读的，请在编辑区域进行文档编辑。

3. 文件资源管理器

此页面显示您的文件和文件夹，保存着您的笔记和资源（图片，pdfs 等）。

🎗️ 小提示：我们推荐您使用树导航来导航你的笔记文件。

4.大纲窗口

显示当前笔记的标题大纲，通过点击大纲来跳转到制定区域。

5.日历窗口

每天记录的快捷入口，如果所选时间不存在一个记录，将会自动创建记录。

6.树导航视图

这里显示笔记的分级视图，类似于文件夹结构在文件系统中的显示方式，点击就可以快速浏览笔记。

7.外链导航窗口

显示编辑窗笔记所有的外链。 导航窗口
